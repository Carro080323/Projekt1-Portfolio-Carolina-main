HTMLs historia

Utforska webbens utveckling och tillgänglighet på webben.

ARPANET

Internet har sina rötter i militära projekt, som ARPANET, utvecklat under 1960-talet av det amerikanska försvaret. Syftet var att skapa ett robust kommunikationssystem som kunde fungera även om delar av nätverket gick ner. Detta lade grunden för det moderna internet.

Tim Berners-Lee och webben

Webben föddes i början av 1990-talet när Tim Berners-Lee, en brittisk ingenjör, utvecklade World Wide Web (WWW). Han arbetade på CERN för att skapa ett sätt för forskare att enkelt dela och få tillgång till information över internet. Hans fokus var att bygga ett system där dokument kunde kopplas samman genom länkar, vilket gjorde det lättare att navigera mellan olika sidor och resurser.

W3C (World Wide Web Consortium) grundades 1994 och tog senare över ansvaret för att utveckla och standardisera webben. Detta säkerställde att den kunde växa och utvecklas på ett enhetligt sätt.

World Wide Web och W3C

World Wide Web (WWW) är ett system av webbplatser och webbsidor kopplade genom hyperlänkar. Internet är infrastrukturen som möjliggör dataöverföring, medan WWW är det visuella gränssnittet som användare interagerar med för att navigera och ta del av innehåll online.

Organisationen W3C samlar företag, forskningsinstitutioner och andra intressenter för att skapa riktlinjer och specifikationer för webbstandarder, såsom HTML, CSS och andra webbrelaterade teknologier. Genom att sätta standarder främjas innovation och webben förblir kompatibel och användarvänlig.

JavaScript och DOM

För att göra webbsidor mer interaktiva började utvecklare runt 1995 använda JavaScript, ett programmeringsspråk som gör det möjligt att skapa dynamiska element på webbplatser. JavaScript fungerar tillsammans med DOM (Document Object Model), som är en struktur som representerar HTML-dokumentet och gör det möjligt att manipulera innehållet, stilen och strukturen på en webbsida.

HTML5

HTML5 började utvecklas 2004 och lanserades officiellt 2014. Denna version gjorde det möjligt för utvecklare att skapa avancerade och användarvänliga webbapplikationer utan externa plugins. HTML5 inkluderade nya funktioner som video- och ljudstöd, geolocation och förbättrad formulärhantering.

Responsiv design och webbutveckling

Mellan 2010 och 2019 ökade användningen av mobila enheter, vilket ledde till en stark efterfrågan på responsiv webbdesign som anpassar webbplatser efter olika skärmstorlekar. CSS3 blev populärt för mer visuellt tilltalande layouter och animationer. JavaScript-bibliotek som jQuery, Angular och React förenklade skapandet av komplexa webbapplikationer.

År 2019 undertecknade WHATWG och W3C ett avtal om att samarbeta kring HTML5 för att säkerställa en enhetlig standard, vilket gör webben mer kompatibel över plattformar och enheter.
