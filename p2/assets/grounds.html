Grundinfo (meta)

<!DOCTYPE>

Anger vilken HTML-standard sidan använder så webbläsaren tolkar innehållet rätt.

<html>

Rot-elementet som innehåller hela HTML-dokumentet.

<head>

Innehåller information om sidan, som metadata, länkar och inställningar.

<title>

Bestämmer namnet som visas i webbläsarfliken.

<style>

Låter dig skriva CSS direkt i HTML-filen.

<link>

Används för att koppla externa filer, oftast CSS.

<body>

Allt synligt innehåll på sidan ligger här.

Semantiska byggblock

<main>

Huvuddelen av sidan – det viktigaste innehållet.

<header>

Introdel som ofta innehåller rubrik, logga och navigation.

<section>

Delar upp sidan i tydliga avsnitt med relaterat innehåll.

<footer>

Botten av sidan med info som kontakt eller copyright.

<article>

Fristående innehåll som kan stå för sig själv.

<aside>

Extra innehåll som är kopplat till huvudtexten, ofta sidoinfo.

Text och rubriker

<a>

Skapar klickbara länkar till andra sidor eller platser.

<span>

Används för att styla små delar av text.

<p>

Markerar ett stycke text.

<h1>...<h6>

Rubriker i olika nivåer från störst till minst.

<ol>

Numrerad lista i ordning.

<ul>

Punktlista utan ordning.

<li>

Enskilt listobjekt i en lista.

<strong>

Markerar viktig text med extra tyngd.

<em>

Betonar text genom kursiv stil.

Sidstruktur

<div>

En generell behållare som används för att gruppera innehåll.

CSS – design

Typografi & stil

font-family

Bestämmer vilket typsnitt texten ska ha.

font-weight

Gör text tunn eller fet.

font-style

Ger texten stil som kursiv eller normal.

font-size

Styr hur stor texten är.

text-decoration

Lägger till t.ex. understrykning eller genomstrykning.

text-transform

Ändrar textens storlek på bokstäver (VERSALER/gemener).

Färg & bakgrund

color

Bestämmer textfärg.

background-color

Sätter bakgrundsfärg på ett element.

background-image

Lägger in en bild eller gradient i bakgrunden.

background

Kombinerar färg, bild eller gradient i bakgrunden.

linear-gradient

Skapar en färgövergång mellan flera färger.

Layout (CSS)

position

Bestämmer hur ett element placeras på sidan.

top/right/bottom/left

Styr avstånd från kanterna i layouten.

display

Bestämmer hur element visas (block, flex osv).

justify-content

Justering i flexbox horisontellt.

align-items

Justering i flexbox vertikalt.

flex-direction

Bestämmer riktningen i en flexbox.

gap

Avstånd mellan element.

text-align

Justerar textens placering.

Storlek & spacing

width

Bredd på ett element.

height

Höjd på ett element.

max-width

Maximal bredd ett element kan ha.

min-height

Minsta höjd ett element kan ha.

margin

Yttre avstånd runt ett element.

padding

Inre avstånd i ett element.

margin-inline

Avstånd vänster/höger i layout.

Effekter & detaljer

object-fit

Bestämmer hur bilder anpassas i sin ruta.

aspect-ratio

Styr proportionerna mellan bredd och höjd.

nth-child()

Väljer element baserat på position i listan.
